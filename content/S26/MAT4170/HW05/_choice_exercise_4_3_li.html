• Prove the following two statements about equivalence relations, which are relations that are reflexive, symmetric, and transitive.
    1. [3] Prove that group isomorphism is an equivalence relation.
    2. [2] Prove that an equivalence relation on a set partitions the set, which is to say that we can write a set as \[ S = S_1 \cup S_2 \cup \dots \cup S_n \] where each $S_i \subseteq S$, and $S_i \cap S_j = \emptyset$ if and only if $i \neq j$.
